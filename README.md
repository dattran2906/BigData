Dự án này mô phỏng kiến trúc Data Lake theo mô hình Medallion Architecture gồm 3 tầng dữ liệu:
Bronze Layer – Raw Data
Silver Layer – Cleaned & Processed Data
Gold Layer – Analytics & Aggregated Data
Repo này được dùng để lưu trữ dữ liệu và thực hành các quy trình xử lý trong môn Nhập môn Dữ liệu lớn.
